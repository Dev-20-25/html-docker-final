# HTML Docker Application

This project demonstrates building and pushing a Docker image using GitHub Actions.

## Features
- Simple HTML application served by Nginx
- Automated Docker build and push using GitHub Actions
- Docker Hub integration

## Usage
The Docker image is automatically built and pushed to Docker Hub when code is pushed to the main branch.

## Workflow
1. Push code to main branch
2. GitHub Actions builds Docker image
3. Image is pushed to Docker Hub
